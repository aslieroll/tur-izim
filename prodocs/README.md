# Türkçe ürün ve teknik dokümantasyon (`prodocs`)

Bu klasör Tur İzim için **birincil dokümantasyon** konumudur. Metinler Türkçedir; kod, tablo ve API adları gerektiğinde İngilizce kalır.

## Teslim seti (zorunlu dosya adları)

| Dosya | İçerik |
|--------|--------|
| **PRD.md** | Ürün gereksinimleri |
| **tech-stack.md** | Kullanılan teknolojiler özeti |
| **Plan.md** | Yüksek seviye yürütme planı |
| **DesignSystem.md** | Flutter tasarım sistemi ve UI ilkeleri |
| **Progress.md** | İlerleme özeti |

## Referans ve ek dokümanlar

- `business-rules.md`, `user-flows.md`, `database-schema.md`, `api-contract.md`, `suitability-score.md`, `product-scope.md`
- `premium-travel-pass-ui-guidelines.md`, `stitch-export-screen-map.md`, `google-stitch-urun-ve-tasarim-brief.md`
- `cursor-prompts.md` — Cursor için teknik prompt şablonları (çoğunlukla İngilizce talimat metni)

Kaynak önceliği çelişkilerinde sıra: `PRD.md` → `business-rules.md` → `user-flows.md` → diğerleri (`DesignSystem.md` içindeki ayrıntılı sıra).
